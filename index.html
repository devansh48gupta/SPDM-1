Aim & Objectives

The aim of this initiative is to provide women with better means and resources to face menstruation. Our SPDM is an easy to operate and cost effective machine designed to empower women.


The society has shrouded this topic with so much stigma and taboos that women find it difficult to speak out publicly in this matter. One of the main objectives of the SPDM is to break this system and to make this topic of menstruation more mainstream so that parents could talk to their daughters more freely, friends could support each other and women would not feel ashamed of something that is natural to them and parts of their life


In urban areas, over the years this issue has been raised and talked about more in conversations and we feel that this the right time to introduce the SPDM so that the awareness that is being created can be capitalised on and women and girls all over the country could break this cloud of stigma. In educational institutions especially in rural areas, women over a long period of time have complained about the lack of infrastructure ranging from unhealthy toilets (or even no washrooms in some cases!) to unavailability of sanitary products. With the SPDM we wish to eliminate this issue starting from our own campus where the sanitary pads will be made available to the students easily and in convenient places so as to reduce any discomfort to the user.

Code Generation


Each customer will be provided with a 5 digit random code. The code in the initial stages will be generated randomly by the system and a customer will receive a code in return of a payment.There are a total 3125 combinations possible with the current arrangement and we may be able to expand upon that through a collaboration with PayTM which is detailed later in this proposal.


There are three types of memory available in Arduino :-

Flash Memory

Flash memory is used to store your program image and any initialized data. You can execute program code from flash, but you can't modify data in flash memory from your executing code.To modify the data, it must first be copied into SRAM.

SRAM

Static random-access memory is a type of random-access memory that uses latching circuitry to store each bit. SRAM is volatile memory; data is lost when power is removed.

EEPROM

Electrically Erasable Programmable Read-Only Memory (EEPROM) is a steady, non-volatile memory storage scheme that is used for storing minimal data quantities in computer and electronic systems and devices, such as circuit boards. This data may be stored, even with no enduring power source, as device pattern or calibration table.


These codes will be stored in the Arduino using EEPROM. This is because in EEPROM as it is non-volatile data will remain stored even without a power supply. As data canâ€™t be modified in flash memory directly and in SRAM there is a need for a continuous power supply or else the data will be lost. Hence EEPROM provides the most suitable means of storing the data to meet our needs.

Working

After the customer makes the necessary payments the user will receive their 5 digit random code. The user will have to enter the code through the keypad provided, the system will then check for the code in the memory. Each code can be accessed once, so if a match for the code is found in the memory a message will be displayed on the screen stating that correct code was entered and the pad can now be taken. After that the code will be erased from the memory and it will become invalid.


So if that code is used again an error message will appear on the screen. If a code is entered and it is not found in the memory by the system an error message will be displayed on the screen to inform the user and will ask for a correct code to be entered.


The internal structure initially will consist of one common rod and three compartments to hold the sanitary pads. The common rod will act as the main rotating part of the machine. There will be spokes in each compartment that will be separating each sanitary pad. After a valid and operational code is entered by the customer the rod will rotate making the compartment move by a unit distance making the pad fall down into the opening provided so that the user can collect it.

Our team reached out to PayTM so as to ensure a smoother code generation and payment option. If this partnership works, we will be provided a QR code by PayTM which will be pasted on the machine or other parts of the campus for payments, which will directly go to the linked bank account. After the payment along with the transaction ID the user will also receive a random code which can be used in the SPDM.
But for that partnership to be established PayTM will require certain documents and PAN card details so as to ensure our validity and credibility. A certification from the university regarding the authenticity of our work and machine will be essential in setting up this partnership.

Cost & Revenue


Once introduced our vending machine will be one of the cheapest in the market as we will be providing a digital machine in the price of a mechanical one i.e. around Rs.3000 . Mechanical machines though cheap, are less accurate and have a much smaller capacity and have faults and issues arising which are a little complicated to fix. Our machine provides a simple mechanism which is easy for maintenance and operation. Our operating and other variable costs can be met by sponsorship and advertising revenue.

There are two major areas for that:-

Sticker Marketing

Companies can be approached to display their stickers and advertisements on the body of our machine.

Sanitary Pad

Companies specializing in production and distribution of sanitary pads can be approached with the offer that only their branded pads will be put in our machine and in return they provide us the pads at the production cost.
